# Преобразователь из Ehernet к CunBusFrame

## Структура проекта

```C++
📦TcpClientUsr200
 ┗ 📦Client
 ┃   ┗📜tcpclient.cpp
 ┃   ┗📜tcpclient.h //tcp клиент
 ┃
 ┗ 📦utils
 ┃    ┗ 📜hexspinbox.cpp //виджет hex QSpinBox
 ┃
 ┗ 📜mainwindow.h // отрисовка виджетов
 ┗ 📜mainwindow.cpp
 ┃
 ┗ 📜main.cpp // основная ф-ия
 
```

### Запуск программы
- Склонить проект
- Открыть в Qt creator .pro файл
- Скомпилировать проект
